# 自动和弦在线Web程序 - 项目进度报告

## 项目概述

自动和弦在线Web程序是一个交互式网页应用，允许用户通过键盘按键或按键组合来播放不同的音乐和弦。该应用面向音乐学习者、作曲家和音乐爱好者，为他们提供一个便捷的和弦演奏工具。
